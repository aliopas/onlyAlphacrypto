# Phase 0: Fix Remaining P0 Issues

## Context
The Lead Architect identified 2 blockers and 1 code-style violation in Phase 0. All three fixes are isolated, low-risk, and touch exactly 3 files.

---

## Fix 1 — Schema: Add `eventType` & `eventSeverity` to `rawNewsBuffer`

**File:** `backend/src/models/market.model.ts` (line 61)

**Problem:** The `rawNewsBuffer` table is missing `eventType` and `eventSeverity` columns. These fields are returned by `generateLightweightTriage()` (confirmed at `openai.service.ts:172-173`) but cannot be persisted because the schema doesn't have them. This breaks Phase 4 (buildTemporalPattern) and Phase 7 (aiWorkflow).

**Change:** Add two columns after `relevanceScore` (line 61):
```ts
eventType: varchar('event_type', { length: 50 }),
eventSeverity: integer('event_severity'),
```

**Migration:** Run `npx drizzle-kit push` from `backend/` to apply the schema change to the database.

---

## Fix 2 — Triage Engine: Persist `eventType` & `eventSeverity`

**File:** `backend/src/crons/triageEngine.cron.ts` (lines 61-67)

**Problem:** The `db.update().set()` block only saves 3 of 5 fields returned by triage. `eventType` and `eventSeverity` are dropped.

**Change:** Add the two missing fields to the `.set()` object:
```ts
await db.update(rawNewsBuffer)
    .set({
        relevanceScore: scoredItem.relevanceScore,
        sentimentHint: scoredItem.sentimentHint,
        symbolMentions: scoredItem.symbolMentions,
        eventType: scoredItem.eventType,
        eventSeverity: scoredItem.eventSeverity,
        processed: true
    })
    .where(eq(rawNewsBuffer.id, item.id));
```

## Fix 3 — Remove `any` type in TerminalWire

**File:** `frontend/src/features/terminal/components/TerminalWire.tsx` (line 70)

**Problem:** `{radarSignals.map((item: any, i: number) => {` violates the zero-`any` rule.

**Solution:** The `RadarSignal` type is already imported at line 5. Remove the explicit `any` type annotation — TypeScript will infer the correct type from the `radarSignals` prop which is typed as `RadarSignal[]`.

**Change:** Replace:
```tsx
{radarSignals.map((item: any, i: number) => {
```
With:
```tsx
{radarSignals.map((item, i) => {
```
