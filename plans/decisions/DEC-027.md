# DEC-027: SEO & Meta Tags Remediation Phase

**Date:** May 13, 2026
**Status:** APPROVED
**Scope:** Frontend Only

## Decision

13-task remediation to fix platform-wide SEO and meta tag gaps discovered during full audit.

## Key Findings from Audit

1. Home page (`/`) has zero metadata export — highest traffic page
2. Airdrop detail pages have no JSON-LD structured data — missing rich results
3. Airdrop 404 pages not marked `noindex` — crawl waste
4. 6 static pages missing OG tags and canonical URLs
5. `SITE_URL` hardcoded in 3 files instead of importing from constants
6. `twitter:site` missing platform-wide
7. Scorecard missing from sitemap and has no structured data
8. Terminal index page has no metadata

## Scope

- 13 files modified (all frontend)
- 0 backend changes
- 0 new npm packages
- 1 optional new file (alpha OG image)

## Phase Document

See: `plans/SEO-META-TAGS-PHASE.md`

## Exit Gate

- Lighthouse SEO score >= 95 on key pages
- All public pages have unique title + description + canonical + OG tags
- Zero hardcoded SITE_URL outside constants.ts
